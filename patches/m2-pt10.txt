
                       EarthBound/Mother 2 Tradução de Fãs
                           Português Brasileiro v 1.0
                             Lançado 04 Abril 2015

Esta é uma tradução não oficial e gratuita de EarthBound (ou MOTHER 2), um RPG
japonês lançado para o Super Famicom em 1994. 

Dê o seu apoio à série adquirindo produtos oficiais, como o jogo para Wii U, e
também a tradução oficial, se um dia esta for lançada.

Para mais detalhes sobre esta tradução e o projeto, acesse:

                              http://www.earthboundbrasil.com

==================================

COMO USAR O PATCH DA TRADUÇÃO:

  1. Extraia o conteúdo deste arquivo para uma pasta no seu computador. Você
     terá os seguintes arquivos:
     
    * ebbr.ips
	* ebbr[!].ips
    * ebbr.txt (que é este arquivo)


  2. Depois, você precisará de uma ROM de EarthBound. Isso pode ser
     considerado pirataria, então você terá que procurar sozinho. Sites de
     busca são úteis para esse tipo de coisa. Precisa ou de um rom com
	 cabeçalho ou sem cabeçalho, mas é melhor sem.
     
     Assim que você tiver a ROM, certifique-se de que não esteja zipada. O
     arquivo deve ter 3 MB. Coloque esse arquivo na mesma pasta que está
     o arquivo ebbr.ips (se está usando ROM com cabeçalho) ou ebbr[!].ips
	 (se está usando ROM sem cabeçalho).

     Se tiver programa que gera MD5 CHECKSUM, o MD5 do ROM com cabeçalho 
	 tem que ser:
     ABE493B665F7467000BCF8C373B323FD
	 
	 E o MD5 do ROM sem cabeçalho:
	 A864B2E5C141D2DEC1C4CBED75A42A85

     
  3. Se você estiver usando Windows, use Lunar IPS para aplicar o patch:
     http://fusoya.eludevisibility.org/lips/
  
     Se você estiver usando Mac, use MultiPatch para aplicar o patch:
     http://projects.sappharad.com/tools/multipatch.html
     
     Se você estiver usando Linux, use JIPS para aplicar o patch:
     http://zerosoft.zophar.net/jips.php
      
     Note que o processo pode demorar algum tempo em um computador mais
     antigo ou se estiver sendo executado a partir de uma mídia externa.
     Em geral, deve levar apenas alguns segundos para o patch ser aplicado.
	 
	 O MD5 do ROM patcheado:
	 A5EE470AD753D22A1C493C7B4BAE6779

     
  4. Quando o patch tiver sido aplicado corretamente, você estará preparado
     para jogar MOTHER 2 em Português! Use o seu emulador de Super Nintendo
     preferido, ou, se tiver os hardwares necessários, você poderá jogar nos
     consoles ou portáteis que quiser.
